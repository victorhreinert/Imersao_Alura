### 💾 Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:victorhreinert/Imersao_Alura.git
```

Entre no diretório do projeto

```bash
  cd Spotify-Imersao-Alura
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
